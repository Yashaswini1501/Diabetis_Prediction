# 🩺 Diabetes Prediction using Logistic Regression

## 📌 Project Overview
This project uses Logistic Regression to predict whether a person has diabetes based on medical features such as glucose level, BMI, age, etc.

---

## ⚙️ Steps Performed

### 1. Data Exploration (EDA)
- Analyzed dataset structure, distributions, and correlations
- Identified patterns and outliers

### 2. Data Preprocessing
- Handled missing/invalid values
- Scaled features using StandardScaler
- Split dataset into training and testing sets

### 3. Model Building
- Built a Logistic Regression model using scikit-learn
- Trained the model on the dataset

### 4. Model Evaluation
- Evaluated using accuracy, precision, recall, F1-score, and ROC-AUC
- Visualized performance using ROC curve

### 5. Interpretation
- Analyzed feature importance using model coefficients
- Identified key predictors like Glucose and BMI

### 6. Deployment
- Created a Streamlit web app for real-time predictions
- Users can input values and get predictions instantly
